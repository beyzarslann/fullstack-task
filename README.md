Kullanılan Teknolojiler <br />
NestJS <br />
React (Vite) <br />
Firebase Cloud Messaging (FCM) <br />
Ethereum Mainnet (WebSocket) <br />

Backend klsöründe bağımlılıkları yükleyin <br />
npm install <br />
.env dosyanıza Alchemy adresinizi ekleyin <br />
Firebase'den aldığınız serviceAccountKey.json dosyasını ana dizine ekleyin <br />
Uygulamayı başlatın <br />
npm run start:dev <br />

Frontend klsöründe bağımlılıkları yükleyin <br />
npm install  <br />
src/firebase.js dosyasına Firebase Web Config bilgilerinizi ekleyin <br />
Başlatın : <br />
npm run dev <br />
